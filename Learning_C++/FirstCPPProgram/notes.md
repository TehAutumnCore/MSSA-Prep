# C++ Syntax
```Cpp
std::cout << "Hello, world << std::endl;

std -> standard namespace (contains C++ standard library components)
:: -> Scope resolution operator(access members inside a namespace or class)
cout -> console output stream object (used to print to the console)
<< -> stream insertion operator(inserts data to the output stream on the left)
"" -> string literal(text inside double quotes)
std::endl; -> stream manipulator: inserts new like and flushes the stream(forces immediate display)
```
